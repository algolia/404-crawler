# 404 Crawler

A command line interface to crawl pages from sitemap urls and all their sub-path.

![Screenshot](https://github.com/algolia/404-crawler/blob/main/README.png)

## Get started

### Install

```sh
pnpm i
```

### Build

```sh
pnpm build
```

### Dev mode

```
node dist/cli.js crawl <options>
```
